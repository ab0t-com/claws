Privacy & Security

Your data never leaves your server.

OpenClaw runs on your infrastructure — not ours. Every conversation, every file, every credential stays where you control it. No cloud dependency. No data harvesting. No third-party access.

  • ✓ All data stored locally — nothing phones home
  • ✓ Credentials are isolated per employee — never shared between them
  • ✓ DM access requires explicit pairing codes — no open doors
  • ✓ Optional S3 backup with automatic credential exclusion
  • ✓ HTTPS via automatic Let's Encrypt — zero configuration
🏠
Self-Hosted
Runs on your own server. VPS, home lab, or office hardware. You choose where your data lives.
🔐
Isolated
Each AI employee has separate credentials, sessions, and workspace. No cross-contamination.
🛡️
Encrypted
HTTPS by default. Token auth on every connection. Pairing codes for new contacts.
💾
Backed Up
S3 sync with smart exclusions. Credentials never leave the server. Versioned for recovery.
